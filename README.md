
# Artı Yaşam Tur Yönetim Sistemi

## Proje Amacı
Bu proje, kullanıcıların tur bilgilerini görüntüleyebildiği ve yöneticilerin tur yönetimi yapabildiği PHP tabanlı bir web uygulamasıdır.

## Kullanılan Teknolojiler
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Proje Yapısı
- docs/ -> Gereksinim analizi ve UML dokümanları
- src/ -> Tüm kaynak kodlar
- database/ -> SQL yedek dosyaları

## Kurulum
1. Projeyi bilgisayarınıza indirin.
2. XAMPP veya benzeri bir localhost ortamı kurun.
3. Veritabanını phpMyAdmin üzerinden içe aktarın.
4. Proje klasörünü htdocs içine taşıyın.
5. Tarayıcıdan localhost üzerinden çalıştırın.

## Git Gereksinimi
Proje GitHub / GitLab / Bitbucket üzerine yüklenmelidir ve en az 5 commit içermelidir.
