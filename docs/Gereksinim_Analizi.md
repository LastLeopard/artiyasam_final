
# Gereksinim Analizi Dokümanı

## Proje Adı
Artı Yaşam Tur Yönetim Sistemi

## Proje Amacı
Kullanıcıların tur listeleme, tur detaylarını görüntüleme ve yönetici paneli üzerinden tur ekleme/düzenleme işlemlerini yapabildiği web tabanlı bir tur yönetim sistemi geliştirmek.

## Fonksiyonel Gereksinimler
- Kullanıcılar aktif turları görüntüleyebilmelidir.
- Kategorilere göre tur filtreleme yapılabilmelidir.
- Yönetici giriş sistemi bulunmalıdır.
- Yönetici yeni tur ekleyebilmelidir.
- Yönetici mevcut tur bilgilerini düzenleyebilmelidir.
- Sistem veritabanı bağlantısı ile çalışmalıdır.
- Site ayarları yönetilebilmelidir.

## Fonksiyonel Olmayan Gereksinimler
- Sistem PHP ve MySQL altyapısı ile çalışmalıdır.
- Hata yönetiminde try-catch yapıları kullanılmalıdır.
- Kodlar modüler klasör yapısında tutulmalıdır.
- Arayüz kullanıcı dostu olmalıdır.
