
# UML Diyagramları

## Use Case Diyagramı
Aktörler:
- Kullanıcı
- Yönetici

Kullanıcı İşlemleri:
- Tur listeleme
- Tur detay görüntüleme
- Kategori filtreleme

Yönetici İşlemleri:
- Giriş yapma
- Tur ekleme
- Tur düzenleme
- Site ayarlarını değiştirme
